# Ejecutar el desarrollo

1. Clonar el repositorio
2. Ejecutar ```pnpm install```
3. Tener Nest CLI instalado ```pnpm add -g @nestjs/cli```
4. Ejecutar la base de datos ```docker-compose up -d```

## Stack
* React Router 7 framework mode
* Nest
* MongoDB
